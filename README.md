# Project Title: Rock-Paper-Scissors

## Description:
Rock Paper Scissors is a classic played by a minimum of two individuals as a fair tool for decision making. 
The game is played using hand signs in the form of a closed fist (Rock), flat hand (paper), and a fist with the index finger and index finger extended (paper).

### Game Rules:
Rock Paper Scissors features three basic rules:

- Rock blunts Scissors (Rock wins over Scissors)
- Scissors cuts Paper (Scissors wins over paper)
- Paper covers Rock (Paper wins over Rock)
- A tie results if both players choose the same option (players play another round to break the tie).

### How To Play the Game
1. Run main.py on your terminal (or powershell)
2. Input your name
3. Input your choice: 'R' which stands for Rock, 'S' which stands for Scissors, or 'P' for Paper
4. Wait for result
5. The End.
